## 🚀 在 Unity 中快速设置

### 步骤 1: 使用 SceneManager 挂载脚本

你已经有一个名为 `SceneManager` 的空对象，可以直接使用它挂载所有核心脚本！

1. 选择场景中的 `SceneManager` 对象
2. 添加以下组件：
   - `GameManager` 脚本（位于 `Assets/Scripts/Game/GameManager.cs`）
   - `QwenService` 脚本（位于 `Assets/Scripts/AI/LLM/QwenService.cs`）
   - `EmotionAnalyzer` 脚本（位于 `Assets/Scripts/AI/Emotion/EmotionAnalyzer.cs`）
   - `DialogueManager` 脚本（位于 `Assets/Scripts/AI/Dialogue/DialogueManager.cs`）
   - `MemorySystem` 脚本（位于 `Assets/Scripts/Game/Memory/MemorySystem.cs`）
   - `TTSService` 脚本（位于 `Assets/Scripts/Audio/TTS/TTSService.cs`）

### 步骤 2: 配置 QwenService

1. 在 Inspector 中找到 `QwenService` 组件
2. 确认 API Key 已设置：`sk-c617d9c68fb04b14a19446c1436c1967`
3. 确认模型：`qwen3.6-plus`
4. 确认 API 端点：`https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions`

### 步骤 3: 创建增强版对话 UI（推荐）

使用新的 `EnhancedChatUI`，它已经集成了 LLM、情绪分析和 TTS！

#### UI 结构（按提示词要求）：

```
Canvas（Screen Space - Overlay）
└── ChatPanel（全屏或居中面板）
    ├── Scroll View（聊天区域）
    │   ├── Viewport（带 Rect Mask 2D）
    │   │   └── Content（聊天内容容器）
    │   │       - Vertical Layout Group
    │   │       - Content Size Fitter（Vertical = Preferred Size）
    │   │
    │   └── Scrollbar Vertical
    │
    ├── InputArea（底部输入区）
    │   ├── TMP_InputField
    │   └── SendButton（Button + Text）
    │
    └── EmotionIndicator（可选，情绪显示区）
        ├── EmotionIcon
        └── EmotionLabel
```

#### UI 尺寸与布局：

- **ChatPanel**: 锚点 Stretch（全屏），Padding：左右 40，上 40，下 120
- **Scroll View（聊天区）**: 高度占比 80%，锚点 Top Stretch，Margin：10
- **InputArea（输入区）**: 高度 100，锚点 Bottom Stretch，背景颜色：浅灰
- **Message气泡**: 最大宽度：屏幕宽度的 60%，内边距：10~16，圆角：20

#### 配色方案（现代UI风格）：

- **ChatPanel背景**: #1E1E2F（深蓝灰）
- **用户消息（右侧）**: 背景 #4CAF50（绿色），文字 白色
- **系统消息（左侧）**: 背景 #2D8CFF（蓝色），文字 白色
- **输入框**: 背景 #2A2A3D，文字 白色
- **按钮**: 背景 #4CAF50，Hover 稍亮

#### 创建消息预制体：

1. 创建 `UserMessagePrefab` - 用户消息样式
2. 创建 `AIMessagePrefab` - AI 消息样式（添加 `ChatMessageBubble.cs` 脚本）
3. 创建 `SystemMessagePrefab` - 系统消息样式

4. 添加 `EnhancedChatUI` 脚本（位于 `Assets/Scripts/UI/Chat/EnhancedChatUI.cs`）到 `ChatPanel`，在 Inspector 中分配引用

### 步骤 4: 创建记忆 UI

1. 在 Canvas 下创建一个名为 `MemoryPanel` 的空对象
2. 添加以下子对象：
   - `MemoryContainer`（用于显示记忆碎片）
   - `MemoryProgressBar`（Slider）
   - `ProgressText`（Text）

3. 创建记忆碎片预制体 `MemoryFragmentPrefab`，包含：
   - `Title` Text 组件
   - `Content` Text 组件
   - `Category` Text 组件

4. 添加 `MemoryUI` 脚本（位于 `Assets/Scripts/UI/MemoryUI.cs`）到 `MemoryPanel`，在 Inspector 中分配引用

### 步骤 5: （可选）添加情绪可视化

1. 在 Canvas 下创建 `EmotionVisualizerPanel`
2. 添加 `EmotionVisualizer` 脚本（位于 `Assets/Scripts/UI/EmotionVisualizer.cs`）
3. 配置 EEG 波图表、情绪图标、历史记录等组件

### 步骤 6: 连接引用

在各个脚本的 Inspector 中互相引用：
- `EmotionAnalyzer` 引用 `MemorySystem`
- `DialogueManager` 引用 `QwenService`、`EmotionAnalyzer`、`MemorySystem`
- `EnhancedChatUI` 引用 `DialogueManager`、`EmotionAnalyzer`、`TTSService`
- `GameManager` 引用所有服务

## 🔧 自定义配置

### 添加更多记忆碎片

在 `Assets/Scripts/Game/Memory/MemorySystem.cs` 的 `InitializeDefaultFragments()` 方法中添加新的 `MemoryFragment` 对象。

### 修改 AI 人设

在 `Assets/Scripts/AI/LLM/QwenService.cs` 的 `BuildSystemPrompt()` 方法中修改 System Prompt。

### 调整情绪阈值

在 `Assets/Scripts/AI/Emotion/EmotionAnalyzer.cs` 的 Inspector 中调整：
- `highAnxietyBetaThreshold` - 焦虑检测阈值
- `calmAlphaThreshold` - 平静检测阈值

### 调整记忆进度

在 `Assets/Scripts/Game/Memory/MemorySystem.cs` 的 Inspector 中调整：
- `baseProgressIncrement` - 每次对话的基础进度增加
- `emotionBonusMultiplier` - 情绪参与度的奖励倍数

### 配置 TTS

在 `Assets/Scripts/Audio/TTS/TTSService.cs` 的 Inspector 中调整：
- `voice` - 语音类型（默认 zhixiaobai）
- `speed` - 语速（0.5-2.0）
- `pitch` - 音调（0.5-2.0）
- `volume` - 音量（0-1）

## 🎨 UI 设计建议

### 视觉风格

- **主题**：月球基地科技风
- **颜色**：深蓝、灰色、银色为主，辅以少量橙色警示
- **字体**：等宽字体（科技感）
- **特效**：全息投影、扫描线、粒子效果

### 记忆解锁动画

- 建议使用粒子系统
- 记忆碎片从模糊到清晰的过渡
- 伴随舒缓的音效

### 情绪可视化

- 使用颜色渐变表示情绪强度
- EEG 波图表实时更新
- 历史记录显示情绪变化趋势

## ⚠️ 注意事项

1. **网络连接**：确保设备可以访问互联网（通义千问 API 和 TTS）
2. **API Key 安全**：生产环境建议使用加密存储，不要硬编码在脚本中
3. **性能优化**：在移动端运行时注意监控内存和 CPU 使用
4. **保存进度**：游戏进度会自动保存到 PlayerPrefs 中
5. **TextMeshPro**：确保项目中已导入 TextMeshPro 资源包
